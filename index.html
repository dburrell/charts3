Simple, Free and Open

J-Charts gives you online charting that is simple, free and open.

require 'redcarpet'

Simple, but powerful

Start with only a couple of lines of code, or see the documentation for how to set every detail to your liking. (Not into coding? That's OK! Build a chart with our live demo and copy straight from there).

Free, but not cheap

You will never have to pay a cent but you're still getting a highly configurable product with dozens of possible configurations via the live demo, or millions by fine tuning through the built in javascript controls.

Open and secure

This is a very important mantra for J-Charts, and revolves around two points:

1) Free under GNU GPL v2

All the code used for this project is available under GNU GPL v2, so:

  • is completely without cost forever
  • is available for personal or professional use
  • can be entirely reviewed inside & out by your IT/IS teams

2) Runs client-side

This project runs on pure javascript, so it has access only to data you explicitly provide it. J-Charts is a visualisation tool, not a tool that tries to access your data in any way, nor could it.


Current Version

The current version of J-Charts is still pre-release while we polish some edges, but it should already work perfectly well in your organisation. This release includes:
  • Bar & Scatter Charts
    • Including multiple series & stacking options
  • Pie, Donut & Polar Charts
    • Multiple series for donut
  • Customisable colour scheme
  • Theme files for preloaded colours/borders/etc
  • Automatic conversion from data tables
  • Function for post-conversion data addition
  • Function for post-conversion data adjustment
  • Tooltips displaying data for each record or series
